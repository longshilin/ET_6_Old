详细描述

You can implement this interface to create a dynamic "options container". Add the container to SRDebugger via the SRDebug API.

When the container is added via the API, the initial set of options will be fetched via GetOptions. Options that are added or removed after this point must fire the OptionAdded and OptionRemoved events in order for those options to be added/removed from the debug panel. If you do not intend to fire any events (i.e. this is a static container) then IsDynamic should return false.
