◆ EnableWorldSpaceMode()

RectTransform SRDebugger.Services.IDebugService.EnableWorldSpaceMode ( )

ADVANCED FEATURE. This will convert the debug panel to a world space object and return the RectTransform. This can be used to position the SRDebugger panel somewhere in your scene. This feature is for advanced users only who know what they are doing. Only limited support will be provided for this method. The debug panel will be made visible if it is not already.

返回
The debug panel RectTransform.
