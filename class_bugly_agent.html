◆ EnableExceptionHandler()

static void BuglyAgent.EnableExceptionHandler ( )
static

Only Enable the C# exception handler.

You can call it when you do not call the 'InitWithAppId(string)', but you must make sure initialized the sdk in elsewhere, such as the native code in associated Android or iOS project.

Default Level is LogError, so the LogError, LogException will auto report.

You can call the method BuglyAgent.ConfigAutoReportLogLevel(LogSeverity) to change the level to auto report if you known what are you doing.
